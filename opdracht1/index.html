Progressive enhancement

Progressive Enhancement is een begrip uit de digitale wereld die letterlijk vertaald "progressieve verbetering" betekent. Bij progressive enhancement ontwikkelt een website of webapplicatie eerst een basisversie die volledig werkt voor bezoekers met verouderde browsers, functiebeperkingen, surfen terwijl ze bepaalde technieken zoals JavaScript, afwijkende toestellen en schermresoluties hebben uitgeschakeld, of met een trage verbinding.
Vervolgens, als een browser css aankan, wordt dit ingezet om de pagina aantrekkelijker en overzichtelijker te maken, en datzelfde geldt met Javascript.

JavaScript kan worden ingezet om bepaalde taken voor een gebruiker te versimpelen. Ook binnen deze omgevingen (CSS, JavaScript) moet er rekening gehouden worden met bepaalde functionaliteiten die in oudere versies niet zullen werken.

"You start by establishing a basic level of user experience that all browsers will be able to provide when rendering your web site, but you also build in more advanced functionality that will automatically be available to browsers that can use it." -W3C

Elektrische fiets

HTML - basis

Een gewone fiets heeft enkel een basis, zonder extra functionaliteiten.

CSS - styling

Wanneer de styling wordt toegpast, krijgt de fiets een meerwaarde voor gebruikers.

JavaScript - gedrag

Bij het toevoegen van het gedrag, de elektrische motor in dit geval, zal de fiets makkelijker worden in gebruik.
